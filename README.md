DNA Sequence Analysis using Python & Pandas 🧬

This is a beginner bioinformatics project where DNA sequence data from NCBI was analyzed using Python and Pandas in Google Colab.

The project includes:

reading biological sequence data
converting FASTA sequences into CSV format
analyzing nucleotide composition
calculating GC content
filtering and sorting DNA sequences
Features
DNA sequence length analysis
Nucleotide counting (A, T, G, C)
GC content calculation
Sequence filtering
Sorting based on GC percentage
CSV dataset generation from FASTA file
Technologies Used

Pandas
